## üöÄ Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/Hrishaan1/brewhousecafesample.git
   cd brewhousecafesample
   ```

2. Open `index.html` in your browser:
   ```bash
   open index.html  # macOS
   # or
   xdg-open index.html  # Linux
   # or use your preferred browser
   ```

3. Use Live Servers (like the one in VSCode):
    ```bash
    python3 -m http.server 8000
    # open http://localhost:8000 in your browser
    ```

## ‚ú® Customization Guide

### Images
Replace placeholder images in the following sections:
- Hero section background
- About section image
- Menu item images
- Gallery images
- Testimonial profile pictures

### Content
1. **Menu Items**
   - Update menu items in the menu section
   - Modify prices and descriptions
   - Add or remove menu categories

2. **Contact Information**
   - Update address in the contact section
   - Modify business hours
   - Replace map embed code

3. **Form Integration**
   - Replace Formspree endpoint in the contact form
   - Add custom form validation if needed
   - Modify form fields as required

4. **Social Media**
   - Update social media links in the footer
   - Modify or add new social icons
   - Update contact email

### Styling

- Background: `#F9FAFB`
